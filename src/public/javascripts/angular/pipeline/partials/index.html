LDVM visualizations by XRG, Charles University in Prague

The goal of this application is to provide visualizations of Linked Data datasets using principles of Linked Data Visualization Model (LDVM). You can use it with your own SPARQL Endpoint.

The Linked Data Visualization Model (LDVM) is an adaptation of the general Data State Reference Model (DSRM) for the specifics of the visualization of RDF and Linked Data.
